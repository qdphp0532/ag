# 快速开始

## 环境要求

- Python 3.10+
- 已配置 `.env`（见下方）

## 安装

```bash
pip install -r requirements.txt
```

## 环境变量（.env）

| 变量 | 必填 | 说明 |
|------|------|------|
| `OPENAI_API_KEY` | 是 | OpenAI API 密钥 |
| `OPENAI_BASE_URL` | 否 | 兼容端点（代理等） |
| `LLM_MODEL` | 否 | 模型名，默认 `gpt-4.1-mini` |
| `LANGSMITH_TRACING` / `LANGSMITH_API_KEY` | 否 | LangSmith 追踪 |
| `APP_NAME` / `DEBUG` | 否 | 应用名、调试开关 |

## 运行

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

- 文档：<http://localhost:8000/docs>
- 健康检查：`curl http://localhost:8000/health`
- 调用 Agent：`curl -X POST http://localhost:8000/api/v1/invoke -H "Content-Type: application/json" -d '{"query":"Where did Harrison work?"}'`
- 调用工作流：`curl -X POST http://localhost:8000/api/v1/workflow -H "Content-Type: application/json" -d '{"query":"Where did Harrison work?"}'`

## 测试

```bash
pytest tests/ -v
```

## 常用命令

```bash
# 仅运行健康与 API 测试
pytest tests/api/ -v

# 带覆盖率（需安装 pytest-cov）
pytest tests/ -v --cov=app --cov-report=term-missing
```
