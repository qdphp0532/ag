# 快速开始

## 环境要求

- Python 3.10+
- 建议使用独立虚拟环境（`venv`、`conda`、`uv` 均可）
- 当前依赖组合已在 Python 3.12 下验证
- 已配置 `.env`（见下方）

## 安装

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

若你习惯 `conda`，也可以使用：

```bash
conda create -n ag python=3.12 -y
conda activate ag
pip install -r requirements.txt
```

## 环境变量（.env）

| 变量 | 必填 | 说明 |
|------|------|------|
| `OPENAI_API_KEY` | 是* | OpenAI API 密钥（使用 openai 厂商时必填） |
| `OPENAI_BASE_URL` | 否 | OpenAI 兼容端点（代理等） |
| `LLM_MODEL` | 否 | OpenAI 模型名，默认 `gpt-4.1-mini` |
| `DEFAULT_LLM_PROVIDER` | 否 | 默认大模型厂商：`openai` / `qwen` / `deepseek` / `doubao`，默认 `openai` |
| `QWEN_*` / `DEEPSEEK_*` / `DOUBAO_*` | 否 | 其他厂商配置，详见 [06-大模型多厂商配置](./06-大模型多厂商配置.md) |
| `LANGSMITH_TRACING` / `LANGSMITH_API_KEY` | 否 | LangSmith 追踪 |
| `APP_NAME` / `DEBUG` | 否 | 应用名、调试开关 |
| `DATABASE_URL` | 否 | 数据库 URL，默认 `sqlite:///./data/app.db` |

## 运行

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

- 文档：<http://localhost:8000/docs>
- 健康检查：`curl http://localhost:8000/health`
- 调用 Agent：`curl -X POST http://localhost:8000/api/v1/invoke -H "Content-Type: application/json" -d '{"query":"Where did Harrison work?"}'`

## 测试

```bash
python -m pytest tests/ -v
```

## 常用命令

```bash
# 仅运行健康与 API 测试
python -m pytest tests/api/ -v

# 带覆盖率（需安装 pytest-cov）
python -m pytest tests/ -v --cov=app --cov-report=term-missing
```
